# Bob Bootcamp Builder - Single Slide Summary

## Objective
Automated creation of customized Bob training bootcamps (Bobathons) via guided workflow

## Bob Mode & Workflows
- **Bobathon Builder Mode**: Interactive Q&A captures client requirements
- **Workflows**: Template customization → Lab adaptation → Validation → Packaging

## Tools
- Structured questionnaire for client context
- Auto-generation from responses
- Consistency validation across materials
- Export/packaging for delivery

## Resources
- `bobathon-config.yaml` template (central config)
- 3 progressive labs (basic → advanced → client-specific)
- Schedule templates (3hr/6hr/12hr formats)
- Reference materials (cheat sheets, troubleshooting, examples)

## Deliverables
1. Customized client configuration
2. Tailored lab exercises (3)
3. Detailed schedule/agenda
4. Resource package (guides, support materials)
5. Validation report
6. Ready-to-deliver bobathon kit

## Format
6-hour default: Morning (intro/basics) + Afternoon (client-specific) | 150+ min hands-on | 42% client-focused
